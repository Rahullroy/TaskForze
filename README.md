# TaskForze

TaskForze is a full-stack task management web application built using Node.js, Express.js, EJS, MongoDB, HTML, CSS, and JavaScript. It helps users organize daily tasks with a clean interface and essential productivity features.

---

## Features

- Create, Update, Delete Tasks (CRUD)
- Mark tasks as Complete / Pending
- Task Priority (Low / Medium / High)
- Due Date Management
- Search Tasks
- Progress Tracking
- Dashboard Statistics
- Dark Mode
- Responsive UI

---

## Installation

```bash
git clone https://github.com/Rahullroy/TaskForze.git
```

```bash
cd TaskForze
```

```bash
npm install
```

Create a `.env` file

```env
atlas_URL=YOUR_MONGODB_CONNECTION_STRING
```

Run the project

```bash
node index.js
```
